# Default central lists

This directory is reserved for conservatively maintained shared rules. It is intentionally empty in the initial release.

Each active entry must have:
- a clear scope and reason;
- reproducible evidence;
- an identified maintainer;
- a review/removal date or continuing justification;
- sufficiently narrow matching to avoid unrelated hosts or URLs.

Sites should download lists externally, validate them, and replace local files atomically. cleanfeed-ng itself performs no network access. Multiple independently maintained list sets may be added as separate subdirectories.
